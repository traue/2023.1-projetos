## Objetivo

Este projeto tem como objetivo apresentar a contrução completa de sistema computacional, ou seja, uma aplicação interativa que pode ser:

- Um jogo
- Uma aplicação web (CRUD)
- Uma aplcação desktop (CRUD)
- Um App compelto
- etc.
